# ExpenseTracker v2.9

Your personal expense tracking application is ready!

## How to Access

Open: `ExpenseTracker-v2_9.html` or `index.html`

## Features

- 📊 Track personal & split expenses
- 💰 Manage income and assets
- 🤝 Track loans and credits
- 📈 View spending analysis with charts
- 🌙 Dark mode support
- 📱 Mobile-responsive design
- 💾 Offline-first (all data stored locally)
- 📥 Export data (XLSX/PDF)

## Data Privacy

All your data stays on YOUR device. Nothing is sent to any server.
